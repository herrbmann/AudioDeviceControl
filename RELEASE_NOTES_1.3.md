# AudioDeviceControl 1.3 - Release Notes

## Neue Features

### Automatischer Profilwechsel per WiFi
Profile können jetzt einem WiFi-Netzwerk zugeordnet werden. Die App wechselt automatisch zum passenden Profil, sobald sich das verbundene WiFi-Netzwerk ändert.

### Sound-Feedback
Optionaler Audio-Feedback bei Profil-Wechseln. Konfigurierbare Lautstärke in den Einstellungen.

### Benachrichtigungen
Push-Benachrichtigungen informieren über automatische Profil-Wechsel basierend auf WiFi-Verbindungen.

### Geräte-Übersicht in Profil-Karten
Jede Profil-Karte zeigt jetzt die aktiven Input- und Output-Geräte direkt an - auf einen Blick sichtbar, welche Hardware aktiv ist.

### Ignorier-Liste pro Profil
Geräte können jetzt pro Profil ignoriert werden. Jedes Profil verwaltet seine eigene Liste ignorierter Geräte für maximale Flexibilität.

### Globale Geräte-Verwaltung
Geräte können komplett aus dem App-Gedächtnis gelöscht werden. Sie erscheinen beim erneuten Anschließen wieder, bleiben aber standardmäßig ausgeblendet.

## Verbesserungen

- Dark Mode Optimierungen für bessere Lesbarkeit
- Zentrierte Lautstärke-Einstellung im Sound-Feedback-Bereich
- Verbesserte UI-Konsistenz
